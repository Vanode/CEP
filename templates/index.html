ABOUT THE PROJECT


This website offers an interactive, web-based approach for American Sign Language communication and learning. Created using cutting-edge HTML, CSS, and JavaScript for frontend & TensorFlow, Flask and OpenCV for backend. It puts accessibility first and brings sign language technology to everyone.

  • Purpose: Promote inclusivity and everyday communication
  • Features: Smooth UI, webcam-enabled sign interpreter, instant feedback
  • Upcoming: Real machine learning translation, extended sign support
